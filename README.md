# thallesmaia.com

Personal portfolio site for Thalles Maia — Software Developer.

Live at [thallesmaia.com](https://thallesmaia.com).

## Structure

- `index.html` — page markup and content
- `css/styles.css` — styles
- `assets/` — static assets (favicon, etc.)

## Development

This is a static site with no build step. Open `index.html` directly in a browser, or serve the directory locally:

```sh
npx serve .
```
